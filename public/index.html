What's new in v3.0 — "Redesign & AI"

  • Redesigned UI — cleaner topbar, new result cards with favicon chips, softer shadows, calmer hero.
  • AI mode (opt-in, client-side) — enable in Settings. A tiny model runs entirely in your browser. Chat grounded on your current results. Zero server calls.
  • Better ranking — per-query IDF, phrase proximity boost, freshness signal, parked/ad-heavy hosts demoted.
  • Faster, bigger index — crawler concurrency 12 → 16, per-host politeness tuned, sitemap prefetch, bloom filter persists dedup across restarts.
  • Command palette — press Ctrl+K anywhere to jump.
  • Pin & hide per result, favorite & block domains, reading-mode view, translate button.
  • New tools — TOTP viewer, DNS lookup, HTTP header viewer, contrast checker, CSV↔JSON, YAML↔JSON, markdown preview, diff side-by-side, stopwatch, countdown, and more.
  • 6 new themes — Carbon Fiber, Blueprint, Paper White, Neon Pink, Mocha, Mint Cream (56 total).
  • Safe-view improvements — more URL rewrites, wider ad/tracker blocklist, top warning banner with live ads-blocked counter.

You can re-open this from Settings → About → What's new.
